Lessons 01-12: Set up + HTML noob

Use <pre/> to preformat your code.

          <style type="text/css">
                pre.code {
                  white-space: pre-wrap;
                }
                pre.code:before {
                    counter-reset: listing;
                }
                pre.code code {
                    counter-increment: listing;
                }
                pre.code code:before {
                    content: counter(listing) ". ";
                }
